Sublime text 进阶操作

XeRnHe
2022-12-02
Sublime-text, 工具

0. 前言

相信 Sublime Text 的安装对于读者来说不算难题，所以本文只介绍 Sublime Text 中读者可能用到的几个功能。

1. 缺省源

1-0. 前言

众所周知，Sublime Text 是没有代码缺省源功能的，但天无绝人之路，这里介绍一种效果相近的方法：

Sublime Snippets

Snippets，意为片段。这个功能可以替你插入一些设定好的文本。

1-1. 片段文件

在 Sublime Text 中选择 Tools -> Developer -> New Snippet... 创建片段文件。

在创建好片段文件之后，发现 Sublime Text 为你提供了一个框架：

说明我们需要更改的部分：

  • 在 ![CDATA] 中，填充我们希望插入的文本；

  • 在带有 <tabTrigger> 的一行，我们需要将该行的代码改为这样的形式：

    <tabTrigger>hello</tabTrigger>

    即将两边的符号删掉，并在两个 <tabTrigger> 之间填入你想设置的触发命令，这里以 hello 为例。

    当你在编写代码时输入了完整的触发命令并且按下 Tab 键时，编辑器会将触发命令替换为我们在上文中所设置的文本。

  • 在带有 <scope> 的一行，我们需要将该行的代码改为这样的形式：

    <scope>source.python</scope>

    并在两个 <scope> 之间填入你希望这个片段生效的范围，这里以 source.python 为例。

    至于一个文件的范围名称，可以通过在该文件中选择 Tools -> Developer -> Show Scope Name 以获得。

1-2. 使用

在编写代码时输入上文设置的触发命令并单击 Tab 键即可。

2. 快捷键

2-0. 前言

本文不对 Sublime Text 的所有快捷键进行介绍，而是仅介绍作者会经常使用的一些快捷键。

2-1. 窗口分屏

Alt+Shift+1 将窗口分为 $1$ 屏，即还原为默认单窗口。

Alt+Shift+2 将窗口左右分为 $2$ 屏。

Alt+Shift+3 将窗口左右分为 $3$ 屏，可以通过更改快捷键设置以实现左部分 $1$ 屏，右部分上下各 $1$ 屏的分屏方式：

选择 Preferences -> Key Bindings 以打开快捷键配置文件。

2-2. 文本编辑

2-2-1. 变量名修改

Ctrl+D 选中光标处文本，继续操作会选中下一个相同的文本。

或者通过 Alt+F3 选中代码中所有与当前所选择的文本相同的文本。

选中所有希望修改的文本后开始修改即可，编辑器会同时更改所有选中的文本。

2-2-2. 括号中选择

Ctrl+Shfit+M 选择光标所在括号中的内容，继续操作向外层继续选中。

2-2-3. 插入新行

Ctrl+G 打开搜索框后输入数字以跳转到相应行。

Ctrl+Enter 在当前行的下一行插入新行。

以上即为作者目前使用 Sublime Text 编辑器所使用的所有技巧。
