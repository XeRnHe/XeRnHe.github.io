# 必修 第一册

## 第 1 章  认识化学科学

### 第 2 节  研究物质性质的方法和程序

物质的物理性质包括：颜色、状态、气味、熔沸点、密度、硬度、溶解性、挥发性、导电性、导热性、延展性、吸水性等。

物质的化学性质包括：氧化性、还原性、酸碱性、稳定性等。

#### 钠

钠单质的物理性质：银白色，有金属光泽；$\rho(水)>\rho(钠)>\rho(煤油)>\rho(无水乙醇)$；熔点低于 $100^\circ C$；质地柔软，可用小刀切割；导电性、导热性良好。

钠单质的化学性质：

- 还原性极强，几乎能和所有常见的非金属单质和酸、水发生反应。

- 常温下缓慢氧化：切开钠块，银白色表面迅速变暗，生成白色的氧化钠。

    $\ce{4Na + O2 = 2Na2O}$.

- 加热时剧烈燃烧：钠在空气中燃烧，火焰呈黄色，生成淡黄色的过氧化钠。

    $\ce{2Na + O2 = Na2O2}$.

- 与水反应：浮熔游响红。

    $\ce{2Na + 2H2O = 2NaOH + H2 ^}$.

- 与酸反应：比与水的反应更加剧烈，可能爆炸。

- 与醇类反应：比与水的反应更加温和，生成醇钠和氢气。

> 为什么钠与水的反应比钠与醇的反应剧烈？
>
> 答：烷基是推电子基团，氧氢键极性 水大于醇，氢离子电离程度 水大于醇，与钠反应剧烈程度 水大于醇。

- **不易**与溶液中的金属离子发生置换反应，因其投入水中直接与水反应。

- 可电解**熔融** $\ce{NaCl}$ 制得。

过氧化钠不是碱性氧化物，属于过氧化物。

#### 氯气

氯气的物理性质：黄绿色气体；有强烈的刺激性气味；有毒；密度比空气大；易液化，经压缩可液化成黄绿色液氯；可溶于水，常温下 1 体积水可溶解约 2 体积氯气。

氯气的化学性质：

- 与金属反应：通常氧化到最高价态。

    $\ce{2Fe + 3Cl2 \stackrel{点燃}{=} 2FeCl3}$.

    $\ce{Na + Cl2 \stackrel{点燃}{=} NaCl2}$.

    $\ce{Cu + Cl2 \stackrel{点燃}{=} CuCl2}$.

- 与氢气反应：安静地燃烧，发出苍白色火焰，瓶口会产生白雾（$\ce{HCl}$ 气体结合空气中的水蒸气形成盐酸小液滴）。在**光照**条件下混合会发生剧烈爆炸。

    $\ce{H2 + Cl2 \stackrel{点燃或光照}{=} 2HCl}$.

- 与水反应：生成次氯酸，能使有色物质褪色，并能杀灭细菌，有漂白作用和消毒作用。

    $\ce{H2O + Cl2 = HCl + HClO}$.

- 与碱反应：次氯酸盐为漂白剂、消毒剂的主要成分。制漂白粉时需将氯气通入石灰乳中，因为氢氧化钙微溶，石灰水中含量少。

    $\ce{2NaOH + Cl2 = NaCl + NaClO + H2O}$.

    $\ce{2CaOH + 2Cl2 = CaCl2 + Ca(ClO)2 + 2H2O}$.

- 漂白粉久置于空气中失效：次氯酸酸性弱于碳酸，见光易分解。

    $\ce{Ca(ClO)2 + H2O + CO2 = CaCO3 + 2HClO}$.

    $\ce{2HClO \stackrel{光照}{=} 2HCl + O2 ^}$.

> 酸性 $\ce{H2CO3 > HClO > HCO3-}$，因此次氯酸钠与二氧化碳反应生成 $\ce{NaHCO3}$，而次氯酸钙与二氧化碳反应能够弱酸制强酸是因为生成 $\ce{CaCO3}$ 沉淀。通入过量 $\ce{CO2}$ 后沉淀消失。

某些含氯化合物如次氯酸、次氯酸盐、**二氧化氯**等，具有漂白作用和消毒作用（**氯气无漂白性**）。二氧化氯的漂白能力和消毒能力比氯气强，使用起来比较安全。

### 微项目  探秘膨松剂

![](https://xernhe.oss-cn-beijing.aliyuncs.com/img/20260421112459.png)

正常情况：酵头发酵产生乳酸、醋酸等酸性物质和二氧化碳，二氧化碳使面团疏松。再加入碳酸钠溶液，中和酸性物质的同时再次生成二氧化碳。

酵头发面失败：酸性物质、二氧化碳较少。此时加入碳酸氢钠做膨松剂，生成二氧化碳，但酸性物质少，无法中和碳酸钠，导致馒头发黄。加入食醋可与碳酸钠反应，解决该问题。

![](https://xernhe.oss-cn-beijing.aliyuncs.com/img/20260421113036.png)

> 启发：在作答工业流程相关题目时，可注意加入的物质在不同条件或环境下，功能的不同或功效的差异。

## 第 2 章  元素与物质世界

### 第 1 节  元素与物质分类

![](https://xernhe.oss-cn-beijing.aliyuncs.com/img/20260422135603.png)

混合物有浊液、胶体和溶液等分散系。其中分散质微粒为直径小于 $1nm$ 的分子或离子形成的分散系叫溶液；分散质微粒为直径介于 $1-100nm$ 之间的分子或离子形成的分散系叫胶体；分散质微粒为直径大于 $100nm$ 的分子或离子形成的分散系叫浊液。还可以根据分散质的状态，将浊液分为悬浊液和乳浊液。也可以根据分散剂的状态，将胶体分为气溶胶、液溶胶和固溶胶。

胶体的分散质微粒比溶液的大，能够透过滤纸，但不能透过半透膜。半透膜具有比滤纸更细小的孔隙，只有小分子、离子能够透过，因此可以利用半透膜分离胶体中的杂质分子或离子。这种提纯、精制胶体的方法称为**渗析**。当可见光束通过胶体时，在入射光侧面可观察到光亮的“通路”，这种现象称为**丁达尔现象**或**丁达尔效应**。可用于实验室鉴别一种分散系属于胶体还是属于溶液。胶体还能够**电泳**、**聚沉**。

---

$\ce{NO2}$ 不为酸性氧化物，也无 $+4$ 价氮原子对应的酸根。$\ce{SiO2}$ 是酸性氧化物，但不溶于水，不与水发生反应。

只有比较活泼的金属对应的氧化物，才能与水直接化合生成碱（钾钠钙钡）。绝大多数难溶于水的碱，受热易分解为对应的金属氧化物和水。$\ce{AgOH}$ 及其不稳定，在常温下分解形成氧化银。本质上是金属活动性决定的碱的稳定性。

$\ce{Al2O3,ZnO,Cr2O3,Ga2O3,BeO}$ 等氧化物为两性氧化物。$\ce{SiO2}$ 虽然能与 $\ce{HF}$ 反应生成 $\ce{SiF4}$，但不为两性氧化物，因为 $\ce{SiF4}$ 为共价化合物，不是传统意义上的盐。

### 第 3 节  氧化还原反应

$\ce{2Cu + O2 = [\Delta] 2CuO}$.

$\ce{CuO + H2 = Cu + H2O}$.

$\ce{3CO + Fe2O3 = 2Fe + 3CO2}$.

大部分含氧酸的酸根离子做氧化剂时，溶液的酸性越强，含氧酸根离子的氧化性就越强。

氧化还原反应的应用：

- 研究物质的性质（氧化性和还原性）；
- 实现物质转化与获得能量；以金属的冶炼为例，热还原法、金属置换法、电解法等都是常见的方法。人们可以利用可燃物作为还原剂，选择合适的氧化剂与之发生氧化还原反应获得热能；另外，人们还可以利用氧化还原反应获得电能。

$\ce{MnO2 + 4HCl = [\Delta] MnCl2 + Cl2 ^ + 2H2O}$.

$\ce{2HClO = [hv] 2HCl + O2 ^}$.
