## 高中常见物质别称

### 碱金属化合物

| 化学式 | 别称 | 性质 |
| - | - | - |
|$\ce{NaOH}$|烧碱、火碱、苛性钠|易潮解，不可垫在滤纸上称量。|
|$\ce{Na2CO3}$|纯碱、苏打|水溶液显碱性，常用于去油污。|
|$\ce{NaHCO3}$|小苏打|做发酵粉膨松剂、治疗胃酸过多。|
|$\ce{Na2S2O3}$|大苏打、海波|常作还原剂，工业废水中脱氯，用于碘量法滴定。|
|$\ce{Na2SiO3}$ 水溶液|水玻璃|矿物胶，极性粘合剂，防火材料。试剂瓶不能用玻璃塞，需用橡胶塞。|

### 碱土金属化合物

| 化学式 | 别称 | 性质 |
| - | - | - |
|$\ce{CaO}$|生石灰|碱性干燥剂，与水反应放热。|
|$\ce{Ca(OH)2}$|熟石灰、消石灰|易潮解。|
|$\ce{Ca(OH)2}$ 水溶液|石灰水|悬浊液叫石灰乳。|
|$\ce{CaCO3}$|石灰石、大理石|工业制取CO₂、生石灰、水泥、玻璃的原料。|
|$\ce{CaSO4.2H2O}$|生石膏|加热失去部分水变成熟石膏。|
|$\ce{CaSO4.H2O}$|熟石膏|加水后硬化膨胀，用于固定骨折、制作模型。|
|$\ce{BaSO4}$|钡餐|难溶于酸，医疗上用于胃肠道造影。|
|$\ce{Ca(ClO)2 + CaCl2}$|漂白粉|混	合物，有效成分是 $\ce{Ca(ClO)2}$，需密封避光保存。|
